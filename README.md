# Game Desc.

Last Bell is a zombie survival game that is a 2D choose-your-own-adventure game developed using Python and Pygame, 
in which the outcome is based on the user's decisions. Players must survive a zombie apocalypse by making decisions
that affect the storyline, character relationships, and ending of the game. There are small playable games within the
larger game, such as a space bar game and a maze game.

## Installation

```bash
pip install pygame
```

# NOTES: 
Be sure to have Python 3.11 installed. Please do not run the project on any version more recent than 3.11.
As of the due date, PyGame does not cooperate with versions newer than 3.11, so any other interpreter will raise errors.

## Playing the Game
Button prompts throughout the game can be clicked using the left-mouse button. Pressing the space key will advance 
story dialogue. Player movement in certain scenes can be controlled using arrow keys.
