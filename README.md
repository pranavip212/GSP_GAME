# Game Desc.

Last Bell is a zombie survival game that is a 2D choose-your-own-adventure game developed using Python and Pygame, 
in which the outcome is based on the user's decisions. Players must survive a zombie apocalypse by making decisions
that affect the storyline, character relationships, and ending of the game. There are small playable games within the
larger game, such as a space bar game and a maze game. 

#
